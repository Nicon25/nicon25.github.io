Data Analysis using SQL

Covid-19 Data Exploration

The analysis of COVID-19 data focuses on various aspects and generates insights that allow for a detailed exploration of the pandemic's impact on different continents, countries, and populations. This is achieved through the utilization of SQL queries, data manipulation, and the creation of a view for potential visualization of the analyzed data.

Tools and Technologies Used:

- Data Exploration and Analysis: Joins, CTE's, Windows Functions, Aggregate Functions, Converting Data Types, Creating Views
- SQL Queries: SELECT, WHERE, GROUP BY, ORDER BY, JOIN, SUM, MAX, LIKE, CAST, etc.
- Data Aggregation: Calculating Total Cases, Deaths, Infections by Population, Death Percentage, etc.
- Population and Vaccination Insights: Comparing Total Population vs New Vaccinations, Rolling Sum of Vaccinations
- Advanced Techniques: Using Common Table Expressions (CTEs) for better organization and readability
- Data Visualization Preparation: Creating Views for later analysis and visualization
